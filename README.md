# Minimal SQLite Database
Building a minimal SQLite DB from scratch in C following https://cstack.github.io/db_tutorial/.

This repo includes developer tools specified in the tutorial and some additional ones:
- Linting/auto-formatting via [ClangFormat](https://clang.llvm.org/docs/ClangFormat.html)
- Build tooling via [CMake](https://cmake.org/)
- Behavior-Driven Testing via [RSpec](https://rspec.info/)
